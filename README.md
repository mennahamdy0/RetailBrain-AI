# 🛒 RetailBrain AI

An AI-powered retail shelf monitoring system that uses Computer Vision and Deep Learning to detect products, analyze shelf occupancy, and generate intelligent retail insights.

## 🚀 Features

- Product Detection using YOLOv8
- Retail Shelf Analytics
- Model Performance Comparison
- Smart Recommendation System
- Interactive Streamlit Dashboard

## 🛠️ Technologies

- Python
- Streamlit
- YOLOv8
- OpenCV
- PyTorch
- Pandas
- NumPy
- Matplotlib
